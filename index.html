CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • CSS Sheets must be linked to thhe HTML in order for the changes to be applied
  • Having style implemented externally vs inline or internally, helps us follow "seperation of concerns" making edits easier
  • CSS Rule: [Selector{property:value}] = declaration block
  • Asterisks are used as wildcards to apply effects to all visible elements
  • The card tag is a way to use apply the same style effects to different elements
  • DRY stands for Do not repeat yourself code - it combines style rules and keeps the code neater
